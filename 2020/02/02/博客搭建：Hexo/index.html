博客搭建:Github Pages+Hexo

Posted on 2020-02-02 Edited on 2020-03-08 In 博客搭建

Hexo 博客框架

00 安装 Node.js 

https://nodejs.org/zh-cn/

选择 LTS（长期支持版）

以管理员方式打开cmd

注：runas /user:administrator "cmd/k"

01 查看版本

node -v
npm -v

02 安装cnpm
npm install -g cnpm –reistry=https://registry.npm.taobo.org

注：淘宝镜像源

cnpm -v

03 全局安装hexo-cli

cnpm install -g hexo-cli

hexo -v            查看  hexo-cli项 与 node项

C:\Users\administertor路径下新建文件夹blog

如果将来出错，干掉它！

重新来过

04 cd blog

hexo init      初始化博客

hexo server    启动博客

###本地预览###

http://localhost:4000

操作成功之后会出现博客界面

恭喜你，即将成功

###新建博文###
hexo new “新建文章名称”

05 接下来就需要创作和编辑内容了（请自行学习）

 cd ../..    重新生成，退回至blog 目录


hexo clean

hexo generate

hexo server （重新启动）

本地启动完成

Ctrl+c 结束

通过Github部署至远端

06 新建一个仓库 New repository

Repository name   用户名.github.io

Create repository

07 blog目录下安装Git部署插件

cnpm install hexo-deployer-git --save

08 设置_config.yml文件

notepad _config.yml

type: git
repo: 仓库地址
branch: master

部署至远端

hexo d

09 输入Github账号，密码

注意：需提前git设置用户名和邮箱地址

$ git config --global user.name "John Doe"
$ git config --global user.email "johndoe@example.com"

###拓展——更换博客主题###

例：在blog目录中下载主题

git clone https://github.com/litten/hexo-yilia.git themes/yilia

哎，clone的贼慢，太难了

10 修改_config.yml文件

notepad _config.yml

theme: landscape>>>yilia 更改主题名称

hexo clean
hexo generate
hexo server    

hexo deploy
